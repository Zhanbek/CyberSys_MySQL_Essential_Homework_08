Завдання 1

Вивчіть основні поняття, розглянуті на уроці.

Завдання 2

Створити базу даних з ім'ям MyFunkDB

Завдання 3

У цій базі даних створити 3 таблиці:

У 1-ій містяться імена та номери телефонів співробітників певної компанії.

У 2-й Відомості про їхню зарплату, та посади: головний директор, менеджер, робітник.

У 3-му сімейному становищі, дати народження, де вони проживають.

Завдання 4

Виконайте ряд записів вставки у вигляді транзакції в процедурі, що зберігається. Якщо такий співробітник є відкотіть базу даних назад.

Завдання 5

Створіть тригер, який видалятиме записи з 2-ї та 3-ї таблиць перед видаленням записів з таблиць співробітників (1-ї таблиці), щоб не порушити цілісність даних.
